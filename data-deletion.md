# データ削除手順

最終更新日：2026年6月27日

「メンチン至上主義」（以下「本アプリ」といいます。）の運営者であるKourabo（以下「運営者」といいます。）は、本アプリの利用者（以下「ユーザー」といいます。）が、自身のデータの削除を希望する場合に、本ページに記載の方法によりデータを削除できることを保証します。

本ページは、本アプリのプライバシーポリシー第8条（ユーザーによるデータの削除）の補足として、データ削除の具体的な手順を案内するものです。

---

## 削除可能なデータの種類と削除方法

本アプリに関連して保存されるデータは、保存場所により以下の3種類に分かれます。それぞれ削除方法が異なります。

### 種類1：端末内に保存されているデータ

本アプリは、以下のデータをユーザーの端末内（ローカルストレージ、端末内データ領域等）にのみ保存しています。これらは運営者を含む第三者に送信されることはなく、ユーザーの端末内のみで完結して保存されています。

| 保存されるデータ | 内容 |
|---|---|
| プレイ回数・利用日数 | 利用統計 |
| 問題ごとの正誤履歴・回答時間 | 自己分析機能、復習機能 |
| 復習リスト・お気に入り問題 | 復習・お気に入り機能 |
| キャラクター選択状態 | キャラクター表示の維持 |
| プレイモード別の累計成績 | チャレンジモードの統計 |

**削除方法（以下のいずれか）：**

1. **本アプリ内のデータリセット機能を利用する**
2. **本アプリをアンインストールする**
3. **端末のOS設定から本アプリのデータを削除する**（Androidの場合：設定 → アプリ → メンチン至上主義 → ストレージ → データを消去 等）

これらの方法を実行すると、端末内に保存されたすべてのデータが**即時かつ完全に**削除されます。削除されたデータは復元できませんので、削除前にご注意ください。

### 種類2：広告配信サービス（Google AdMob）に保存されているデータ

本アプリは、Google AdMobを利用して広告を表示しており、Google AdMobは広告配信および広告効果測定のために、広告識別子（IDFA、Android Advertising ID）、端末情報、IPアドレス、広告の表示・操作履歴等を取得する場合があります。

**削除方法（以下のいずれか）：**

1. **端末のOS設定から広告識別子をリセットする**
   - Androidの場合：設定 → Google → 広告 → 広告ID をリセット または 削除
   - iOSの場合：設定 → プライバシーとセキュリティ → トラッキング → 「Appからのトラッキング要求を許可」をオフ
2. **Googleに対してデータ削除を直接請求する**
   - Googleプライバシーポリシー：https://policies.google.com/privacy
   - パーソナライズ広告設定：https://adssettings.google.com/

これらの操作により、Google AdMobに保存されている広告関連データは順次削除または匿名化されます。

### 種類3：課金管理サービス（RevenueCat）に保存されているデータ

本アプリは、RevenueCatを利用してアプリ内課金（買い切り）の購入状態を管理しています。RevenueCatは購入情報、レシート情報、端末情報、IPアドレス、匿名ユーザーID等を取得する場合があります。

なお、本アプリの課金商品は買い切り（非消費型）であり、定期購入（サブスクリプション）ではありません。解約や自動更新の手続きは不要のため、データ削除にあたって事前の解約手続きは必要ありません。

**削除方法：**

1. **本ページ末尾の連絡先メールアドレスまでご連絡ください**
2. メール件名に「**データ削除請求**」と明記してください
3. お問い合わせ内容に、削除を希望される旨と、可能であれば購入時の概略情報（購入日、購入端末のOS等）をお知らせください
4. 受領後、運営者がRevenueCatに対して削除手続きを行います

**保持期間について：**

なお、アプリ内課金の購入情報については、購入の復元機能の提供、会計上の保管義務（税法上の帳簿保存義務等）、消費者保護の観点から、以下のとおり一定の情報が保管される場合があります。これは法令上の要請または運営上の必要性に基づくものであり、削除請求があってもこの期間内は完全な削除ができない場合があります。

- 会計上の保管義務に基づき、**最終取引日から7年間**は購入情報の一部を保管

ただし、購入情報以外のデータ（端末情報、IPアドレス、広告関連情報等）については、削除請求から**30日以内**に削除します。

データ削除に関するご請求・お問い合わせは、以下の窓口までご連絡ください。

- 運営者：Kourabo
- サポート窓口名義：清一色サポート運営事務局
- メールアドレス：**chinitsu.support@gmail.com**

メールには以下の情報をお知らせいただけると、対応がスムーズになります：

2. ご利用の端末OS（iOS / Android）
3. アプリ内課金の有無（種類3の削除請求の場合）

## 削除に関する重要事項

### 削除完了までの目安

| データの種類 | 削除完了までの時間 |
|---|---|
| 端末内データ | 即時（ユーザー操作完了時） |
| Google AdMobのデータ | 端末側のリセットは即時、Google側の処理は数日〜数週間 |
| RevenueCatのデータ | 削除請求から30日以内（購入情報除く） |

### 削除できないデータ

以下のデータは、法令または契約上の理由により、削除請求があっても保持される場合があります：

- アプリ内課金の購入情報（会計法令に基づく7年間の保存義務）
- Apple App StoreおよびGoogle Play Storeにおける決済記録（各ストアの規約による）

### 削除後の影響

データを削除すると以下の影響があります：

- 復習リスト、お気に入り、プレイ履歴、自己分析データ等がすべて失われます
- これらのデータは復元できません
- 端末を変更した際の引き継ぎはできません
- ただし、購入したPremium機能は、同一のApple ID / Google アカウントで「購入の復元」機能により復活します（買い切りのため期限なく利用できます）
